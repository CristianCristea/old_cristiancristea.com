Project Deliverable

Interactive video player module

Project Overview

Build an HTML5 video player using JavaScript and the HTML5 Video API. Using the supplied mockups, video files, and transcript, build an interactive video player that synchronizes the video and the transcript. The transcript should be placed below the video, and should highlight as the video progresses. When a user clicks any part of the transcript it should take them to the appropriate place in the video.

Project details

  • As the media playback time changes, sentences in the transcript highlight.

  • Play and pause buttons

  • As the video plays the playback bar updates

  • As the video plays the current time is displayed and updated
    e.g. 0:10 / 11:34

  • Click anywhere on the playback bar to jump to that part of the video

  • Volume button that lets you mute the sound or turn it on

  • Volume control so viewer can adjust the volume level

  • Fullscreen button
