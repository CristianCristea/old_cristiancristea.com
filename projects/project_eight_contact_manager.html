Project Overview

In this project, you will create an interactive Contact Manager using JavaScript, jQuery, Web Storage API.

Manage contacts. Add, edit, delete, search and store the contacts using the Web Storage API.

Project details

  • Create contact - Name, email and address

  • Store the contact locally on submit.

  • Edit the contact.

  • Delete the contact from the storage.

  • Search through the contacts with instant results during typing.
